# Model-Markowa-wy-szego-rz-du-w-analizie-i-generowaniu-tekstu
Projekt ma na celu zastosowanie łańcuchów Markowa wyższego rzędu (2. i 3. rzędu) do analizy i generowania tekstu na podstawie wybranego korpusu. Projekt wykorzystuje metody przetwarzania języka naturalnego (NLP) oraz elementy uczenia maszynowego do badania sekwencyjnych wzorców języko- wych.
